# 🏢 Asset Leasing Smart Contract (Soroban - Stellar)

## 📌 Project Description
This project implements a basic **Asset Leasing Smart Contract** using Soroban on the Stellar blockchain. It allows asset owners to list their assets for lease and enables users to lease them in a decentralized and trustless manner.

---

## ⚙️ What it does
- Allows an owner to create a leasing listing
- Stores leasing terms (price, owner)
- Allows another user (lessee) to lease the asset
- Tracks leasing state (available or leased)

---

## ✨ Features
- 🔐 Owner authorization for listing creation
- 📄 On-chain lease state storage
- 👥 Lessee assignment
- 🚫 Prevents double leasing
- 🧪 Includes unit tests
